# Project: Workspace Setup

## Goal

Bootstrap and maintain the Candao workspace structure as a standalone AI-readable management entry.

## Scope

In scope:

- Root workspace instructions.
- Project registry and memory files.
- Workspace setup task records.

Out of scope:

- Source code changes inside managed Candao repositories unless a future task explicitly asks for migration or edits.
- Any changes under `E:\Projects\ai-workspace`.

## Source Root

- `E:\Projects\Candao-workspace`

## Outputs

- Workspace structure, project memory files, and setup notes.
