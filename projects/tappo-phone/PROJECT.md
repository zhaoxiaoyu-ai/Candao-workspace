# Project: Tappo Phone

## Goal

Build the Tappo Phone embedded web app for Flutter WebView, covering owner and staff mobile workflows with a polished Tappo visual system.

## Scope

In scope:

- Vue 3 mobile web source code under `source/`.
- Tappo Phone design tokens, components, bridge layer, mock services, and phased page slices.
- Owner, staff, and ordering flow implementation notes.

Out of scope:

- Flutter native shell implementation.
- Real backend integration before API contracts are provided.
- Non-Candao workspace state.
