# T01: Implement Tappo Phone Design Spec v0.1

## Goal

Create the initial Tappo Phone Vue 3 Web layer that implements the approved design spec, bridge boundary, mock data layer, and first page slices.

## Scope

- Vue 3 + TypeScript + Vite app scaffold.
- Tappo visual tokens and base CSS.
- Self-built base components.
- Mock Flutter bridge and mock services.
- Design spec page, owner dashboard slice, staff app center slice, ordering flow slice, order confirm, and order detail.

## Acceptance

- `npm run build` succeeds.
- App runs locally and exposes the first slices via Vue Router.
- Source documents mock-only and bridge-placeholder behavior clearly.
