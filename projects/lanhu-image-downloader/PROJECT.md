# Project: Lanhu Image Downloader

## Goal

Provide a standalone script for downloading Lanhu design images into local folders by project and Lanhu group hierarchy.

## Scope

In scope:

- Lanhu design image download automation.
- Local output directory creation for downloaded images.
- Documentation for safe runtime credential handling.

Out of scope:

- Figma plugins or Design Agent source code.
- Deleting, moving, or modifying existing local files.
- Writing to `E:\Projects\ai-workspace`.

## Source Root

- `E:\Projects\Candao-workspace\projects\lanhu-image-downloader\source`

## Outputs

- Python CLI script and usage notes.

## Context Files

- `PROJECT.md`: project definition
- `RULES.md`: project-specific rules
- `TASKS.md`: task map
- `STATE.md`: current truth
- `HANDOFF.md`: next entry point
- `tasks/<task-id>/task.md`: task definition
- `tasks/<task-id>/output/`: task artifacts
